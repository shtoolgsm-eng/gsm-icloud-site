# 📱 GSM SHTOOL - Professional Mobile Services & Forensic Hub
**Developed by: Eng. Shamshan Shaban**

GSM SHTOOL is an advanced web-based platform designed for mobile technicians and forensic experts. It streamlines device verification, iCloud services, and professional firmware management through a high-performance PHP backend.

---

## 🚀 Key Features & Roadmap

### 1. 💳 Fintech & Payment Integration
* **Crypto Gateway:** Seamless integration with **Binance Pay API** for USDT, BTC, and ETH transactions.
* **Local Gateways:** Manual deposit system for local Yemeni providers (Kuraimi, Al-Najm) with receipt verification logic.
* **Credit System:** User wallet system allowing customers to top up their accounts and pay per service.

### 2. 🎫 Advanced Support Ticket System
* **Issue Tracking:** Comprehensive ticketing system for managing complex bypass and server requests.
* **Status Updates:** Real-time tracking from "Pending" to "Completed" with automated logs.

### 3. 🔔 Real-time Push Notifications
* **Firebase Integration:** Leveraging FCM (Firebase Cloud Messaging) to notify users about order completions and server status directly in their browsers.

### 4. 🔍 Forensic & IMEI Tools
* **Local TAC Database:** High-speed, offline IMEI identification using a built-in Type Allocation Code database.
* **Forensic Ready:** Optimized to handle data structures from **Oxygen Forensic** and **Magnet AXIOM**.

### 5. 📈 SEO & Technical Content
* **Optimized Performance:** Clean, SEO-friendly URLs and Meta-tagging for "iCloud Bypass", "FRP Tools", and "IMEI Check".
* **Knowledge Base:** Integrated technical blog for sharing GSM software solutions.

---

## 🛠 Tech Stack
* **Backend:** PHP 8.x (Modular Architecture)
* **Database:** MySQL (User data) & SQLite (Local TAC lookup)
* **Frontend:** Modern UI/UX with Dark Mode support.
* **APIs:** Binance Pay, Firebase FCM, Google Drive API (Backups).
